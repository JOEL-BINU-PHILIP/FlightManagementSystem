# Flight Management System

A microservices-based Flight Booking application built using **Spring Boot**, **MongoDB**, **Eureka**, **API Gateway**, **Config Server**, and **RabbitMQ**.

This repository contains the complete implementation of:
- Flight Service  
- Booking Service  
- Email Service  
- API Gateway  
- Service Registry  
- Config Server

## Features
- Search for flights  
- View flight details  
- Book a ticket  
- Get booking history  
- Retrieve ticket using PNR  
- Email notifications  
- Centralized config + Service discovery

## How to Run
1. Start **Config Server**  
2. Start **Eureka Server**  
3. Start **API Gateway**  
4. Start **Flight Service**, **Booking Service**, **Email Service**
